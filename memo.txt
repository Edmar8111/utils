CURL keywargs:
    curl -O <https://url> -> Download arquivo via link
    curl -o <novo_nome> <https://url> -> Download de arquivo com nomenclatura alterada
    curl -T <local_arquivo> <ftp://url> -> Upload de arquivos
    curl -x <proxy_url>:<porta>  <https://url> -> Define um proxy para acesso a url
    curl -x <https://url>:<porta> -U <username>:<password> <https://url> -> Define um proxy com login
    curl -H "X-Custom-Header: value" <https://url> -> Efetua a requisição encaminhando um headers
    curl -b "name=value" <https://url> -> Efetua o envio de cookies para a url
    curl -c <name_file.txt> <https://url> -> Salva os cookies no arquivo especificado
    curl -d "name=value" <https://url> -> Efetua o encaminhamento de arquivo via post
    curl --json '{"key":"value"}' <https://url> -> Efetua o envio de um arquivo json


UFW keyargs:
    ativar conexão de portas via ufw -> sudo ufw allow <start_port>:<end_port>/<protocol>
    desativar conexão de porta -> sudo ufw deny <port>/<optional: protocol>
    
    sudo ufw show
    sudo ufw show raw
    sudo ufw status
    sudo ufw delete deny <port>/<optional: protocol>
